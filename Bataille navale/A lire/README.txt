########
## README
########

Ce projet tourne uniquement sous Linux.

Pour compiler le projet, rendez-vous dans le dossier "Bataille navale/c" et entrez la commande suivante : make.
Pour lancer l'application, entrez la commande suivante : ./SoD map.txt
Le fichier map.txt contient les caractéristiques de la carte utilisée pour la bataille navale. Il ne doit pas être modifié.
